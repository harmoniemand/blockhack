wtf is imfuc?

IMFUC is a simple but versatile game about code and movies. You could play it for yourself or with a crowd. You could use it as a funny way for the first interaction with code for someone. You could use it to choose the right employee for your company.

Or you could use it just for fun!

the rules

Its pretty simple. You will see some code. The code referes in any kind to a movie. The game is to find the right movie to the displayed code. Kind of like jeopardy. But very different.

You can play imfuc against each other. In this case, one person or team got 20 seconds to find the right movie. If they make it, they will earn 5 points. If they dont, the other team got 10 seconds to get 4 points. If no team will find the right movie withhin the 30 seconds, they get a hint. Every hint will cost 1 point until 0 points are left. In this case no one will get a point, the answer will be shown and we can go to the next question. Its that simple.

who made this

IMFUC was created by some software-devs from code for heilbronn during a discussion about tree shaking in javascript. Someone said something and they stopped the discussion to start writing code using quotes from movies. Thats it.
